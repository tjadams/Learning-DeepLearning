# ACT
From-scratch/vibe-coded implementation of the Action Chunking with Transformers (ACT) model. ACT allows a low-cost robot to learn difficult tasks (like opening the lid of a container) in the real world with 80-90% success, and only 10 minutes worth of demonstrations.

Paper: https://arxiv.org/abs/2304.13705 + https://arxiv.org/pdf/2304.13705
