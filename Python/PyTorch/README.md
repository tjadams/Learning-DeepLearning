# PyTorch
This folder is about PyTorch for learning purposes.

Manually created a conda env ahead of time.

## Usage - MNIST 
This code is inspired by the PyTorch repo, as an example for learning: https://github.com/pytorch/examples/tree/main/mnist.

Training:
0. conda activate pytorch
1. cd Python/PyTorch/MNIST/
2. python train.py --save-model

Inference:
0. conda activate pytorch
1. cd Python/PyTorch/MNIST/
2. python infer.py
